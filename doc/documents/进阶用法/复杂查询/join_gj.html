join操作

join常用操作有：leftJoin、rightJoin和innerJoin
1.leftJoin介绍： 

/**
 * 左连接
 * @param string $tableName  关联的子表表名，含对子表的重命名如: 'product AS P'
 * @param string  $on on关联信息，如: 'A.id=B.pid'
 */
public function leftJoin($tableName, $on)
leftJoin用法： 

$tester  = new testModel();
$list    = $tester->alias('T')->leftJoin(linkCpModel::getTableName() . ' AS L', 'T.id=L.id')->where(['L.title' => 14])->page(20)->page(2)->all();

2.rightJoin介绍： 

/**
 * 右连接
 * @param string $tableName  关联的子表表名，含对子表的重命名如: 'product AS P'
 * @param string  $on on关联信息，如: 'A.id=B.pid'
 * @return $this
 */
public function rightJoin($tableName, $on)
rightJoin用法： 

$tester  = new testModel();
$list    = $tester->alias('T')->rightJoin(linkCpModel::getTableName() . ' AS L', 'T.id=L.id')->where(['L.title' => 14])->page(20)->page(2)->all();

3.innerJoin介绍： 

/**
 * 内连接
 * @param string $tableName  关联的子表表名，含对子表的重命名如: 'product AS P'
 * @param string  $on on关联信息，如: 'A.id=B.pid'
 */
public function innerJoin($tableName, $on)
innerJoin用法： 

$tester  = new testModel();
$list    = $tester->alias('T')->innerJoin(linkCpModel::getTableName() . ' AS L', 'T.id=L.id')->where(['L.title' => 14])->page(20)->page(2)->all();
