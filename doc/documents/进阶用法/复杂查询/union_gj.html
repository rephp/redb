union操作

union常用操作有：union和unionAll
1.union介绍：
union方法不支持分页，仅简单的将多组查询结果合并在一起显示，要求这些结果结构必须一致。
此方法适用于场景如用户名或者手机或者email等混合都可以登录等特殊场景，可以避免使用or语句，提升查询效率。 

/**
 * union联合查询
 * @param ormModel $model  ormModel对象
 */
public function union(ormModel $model)
union用法： 

$tester  = new testModel();
$list    = $tester->where($where)->union(linkCpModel::db()->where($where2))->all();

2.unionAll介绍：
unionAll方法支持分页，适合分表的数据汇总分页展示，如日志表按月分表后可以使用unionAll汇总分页展示给前端。此方法一样可避开or语句查询，在大量数据检索时可有效命中索引从而提升效率 

/**
 * union all联合查询
 * @param ormModel $model  ormModel对象
 * @return $this
 */
public function unionAll(ormModel $model)
unionAll用法： 

$tester  = new testModel();
$list    = $tester->where($where)->unionAll(linkCpModel::db()->where($where2))->limit(10)->page(1)->all();
