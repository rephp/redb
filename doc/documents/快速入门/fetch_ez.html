查询数据

Redb查询数据主要为按分页查询（按分页数据列表查询+统计数量查询）、查询某条数据


下面我们详细罗列这三种查询方式：

1.统计数量查询 

$tester = new testModel();
$count  = $tester->where($where)->count();
2.查询单条数据 

$tester = new testModel();
$one    = $tester->where($where)->one();
3.查询多条数据（支持分页） 

$tester = new testModel();
$list   = $tester->where($where)->orderBy('id DESC')->limit(20)->page(2)->all();
4.按分页便捷查询数据（含数据+统计总数） 

$tester = new testModel();
$list   = $tester->where($where)->orderBy('id DESC')->limit(20)->page(2)->fetch();
